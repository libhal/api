SysTick Timer#

ARM Cortex-M SysTick Timer#

Defined in namespace hal::cortex_m

#include <libhal-arm-mcu/systick_timer.hpp>

class systick_timer : public hal::timer

SysTick driver for the ARM Cortex Mx series chips.

Available in all ARM Cortex M series processors. Provides a generic and simple timer for every platform using these processor.

Public Types

enum class clock_source : std::uint8_t

Defines the set of clock sources for the SysTick timer.

Values:

enumerator external

Use an external clock source. What this source is depends on the architecture and configuration of the platform.

enumerator processor

Use the clock given to the CPU.
