class pwm#

This class should not be constructed directly.

The user should instantiate a General Purpose or Advanced timer class first, and then acquire a pwm pin through that class.

Public Functions

pwm(hal::unsafe, void *p_reg, pwm_channel_info p_settings)#

Construct generic stm32 pwm driver.

Care should be taken in constructing this outside of a platform specific timer class. If both a platform specific timer class and this object exist, care must be taken to ensure that the drivers do not conflict with each others.

Parameters:

  • p_reg ‚Äì is a void pointer that points to the beginning of a timer peripheral

inline pwm(hal::unsafe)#

Construct pwm uninitialized.

The purpose of this is to send the settings through the initialize function instead, because the channel calculation and pin configuration is done in the constructor and the regular constructor cannot be initialized through the initializer list.

If this constructor is used, it is unsafe to call any API of this class before calling the initialize() API with the correct inputs. Once that API has been called without failure, then the other APIs will become available.
