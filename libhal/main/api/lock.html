inline bool try_lock_for(hal::time_duration p_duration)#

Attempt to acquire a lock and block for a duration of time.

This call will block for the supplied amount of time and then return a status on whether or not the lock was acquired.

Parameters:

p_duration ‚Äì - the amount of time to block the current thread waiting for the lock to be acquired. Note that the precision of the lock‚Äôs timing mechanism may not be as precise as the time duration. Expect the time duration that the thread blocks for to be the rounded down to the nearest precision unit of time. For example, if the precision of the system that manages the lock is 1ms, then providing 2.5ms will result in 2ms waited.

Returns:

true - if the lock was acquired in the time provided.

Returns:

false - if the lock was not acquired in the time provided.
